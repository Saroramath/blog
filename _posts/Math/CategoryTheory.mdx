## Terminal Object and Initial Object
An object T in a category is a terminal object if for every object X in category, there is a unique objet with a homomorphism $\phi: X \to T$. <br/>
An object I in a category is an initial object if for every object X in category, there is a unique objet with a homomorphism $\phi: I \to X$.
